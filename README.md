# Portfólio Neo-Brutalist - Otávio

Portfólio pessoal com design Neo-Brutalist, caracterizado por:
- Cores vibrantes e contrastantes
- Tipografia bold e impactante
- Bordas grossas e sombras marcadas
- Layout direto e sem floreios

## Estrutura do Projeto

```
├── index.html              # Página principal
├── public/                 # Arquivos públicos
├── src/
│   ├── css/               # Estilos
│   ├── js/                # Scripts
│   └── assets/            # Imagens e ícones
└── docs/                  # Documentação
```

## Tecnologias

- HTML5
- CSS3 (Neo-Brutalist Design)
- JavaScript (ES6+)
- Animações CSS e JS

## Como executar

Abra o arquivo `index.html` em seu navegador ou use um servidor local.
